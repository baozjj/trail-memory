# Project Context
1. **项目形态**：Monorepo 结构，包含 `frontend` 和 `backend` 两个目录。
2. **当前进度**：前端已完成 UI 原型，后端代码目前为空。
3. **技术栈**：
   - 前端：Vue 3 + TDesign Mobile + Tailwind CSS。
   - 后端：Node.js + Express + TypeScript + Prisma + SQLite。
4. **数据库**：使用 Prisma 定义 SQLite 结构。

# Task 2: 后端基础设施建设 (地基阶段)
请按顺序执行以下操作，并在每一步完成后向我确认，不要一次性输出全部代码，以保证模块的解耦：

1. **环境初始化**：
   - 在 `backend` 目录下初始化 Node.js 项目。
   - 配置 `tsconfig.json` 以适配 TypeScript 环境。
   - 配置 `package.json`，确保包含必要的 `dev` (开发守护) 和 `build` (TS编译) 脚本。
   - 安装核心依赖：`express`, `cors`, `prisma`, `@prisma/client`, `zod`, `bcrypt`, `jsonwebtoken`。

2. **数据库建模**：
   - 根据本项目需求，建立 `prisma/schema.prisma`。
   - 包含 `User` (用户) 和 `Memory` (印记) 模型。
   - 配置 SQLite 作为数据源。

3. **框架与基础架构**：
   - 配置 Express 基础服务，集成 `cors` 以解决跨域，集成 JSON 中间件。
   - 引入 `express-rate-limit` 进行接口基础防爆破限流。
   - 创建 `src/index.ts` 作为启动入口。
   - 创建 `src/types/` 或 `src/middleware/` 目录，准备好统一的错误处理中间件。
