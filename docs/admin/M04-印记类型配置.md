## 3. 数据模型

```prisma
model ImprintType {
  id        String   @id
  label     String
  coverPath String
  sortOrder Int      @default(0)
  enabled   Boolean  @default(true)
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt
}
```

## 4. 接口

### 管理端

| 方法  | 路径                           | 权限               | 说明                   |
| ----- | ------------------------------ | ------------------ | ---------------------- |
| GET   | `/api/admin/imprint-types`     | imprint-type.read  | 全部，按 sortOrder asc |
| POST  | `/api/admin/imprint-types`     | imprint-type.write | 新建                   |
| PATCH | `/api/admin/imprint-types/:id` | imprint-type.write | 更新（不可改 id）      |

POST Body：

```json
{
  "id": "maolihao",
  "label": "麦理浩径",
  "coverPath": "/imprint-types/maolihao.jpg",
  "sortOrder": 10,
  "enabled": true
}
```

### C 端公开（挂在 `app.ts` 非 admin 路由）

**GET `/api/imprint-types`**

```json
{
  "success": true,
  "data": {
    "items": [
      {
        "id": "xihu-biaoyi",
        "label": "西湖标毅线",
        "coverSrc": "/imprint-types/西湖标毅线.jpg"
      }
    ]
  }
}
```

仅 `enabled=true`，`sortOrder` 升序。字段名 `coverSrc` 与 C 端 `ImprintTypeDefinition` 对齐。
