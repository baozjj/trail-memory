## 5. 环境变量

```env
# 管理端 JWT（必填，与 JWT_SECRET 不同）
ADMIN_JWT_SECRET="change-me-admin"
ADMIN_JWT_EXPIRES_IN="8h"
ADMIN_CORS_ORIGINS="http://localhost:5174"
```

`loadEnv()` 返回扩展字段：`adminJwtSecret`、`adminJwtExpiresIn`、`adminCorsOrigins`。

`app.ts`：

```ts
cors({ origin: [...corsOrigins, ...adminCorsOrigins], credentials: true });
```

## 7. 管理端工程要点

### 7.1 `vite.config.ts`

```ts
server: {
  port: 5174,
  proxy: {
    '/api': { target: 'http://localhost:3000', changeOrigin: true },
  },
}
```

### 7.2 与 C 端对齐的工程初始化

建议以 `frontend/` 为模板裁剪（而非从零选模板），保证 Vite / TS / Prettier / vue-tsc 版本一致：

1. 复制 `frontend/tsconfig*.json`、`vite.config.ts` 结构到 `admin/`
2. 修改 `server.port` 为 `5174`，proxy 保留 `/api` 与 `/uploads`（与 C 端一致）
3. `package.json` 依赖见 §7.3

### 7.3 `admin/src/api/http.ts`

- 参考 `frontend/src/api/axios.ts`（`http` 实例 + 拦截器 + `getApiErrorMessage`）
- Token 键：`trail_admin_token`
- 401 → `router.replace({ name: 'login' })`
