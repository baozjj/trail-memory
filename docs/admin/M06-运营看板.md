## 3. 接口

### GET `/api/admin/dashboard/summary`

```json
{
  "success": true,
  "data": {
    "userTotal": 100,
    "userToday": 3,
    "userTodayDeltaPercent": 50,
    "memoryTotal": 250,
    "memoryToday": 8,
    "publicMemoryTotal": 120,
    "publicMemoryPercent": 48,
    "verifiedUserPercent": 80
  }
}
```

| 字段 | 计算 |
|------|------|
| `userTotal` | `count(User)` |
| `userToday` | `createdAt >= 今日 0 点（服务器时区）` |
| `userTodayDeltaPercent` | 相对昨日同口径，昨日为 0 则返回 `null` 或 100 |
| `memoryTotal` | `count(Memory where deletedAt null)` |
| `memoryToday` | 同上 + 今日创建 |
| `publicMemoryTotal` | `isPublic && deletedAt null` |
| `publicMemoryPercent` | `publicMemoryTotal / memoryTotal`，分母 0 → 0 |
| `verifiedUserPercent` | `isVerified / userTotal` |

### GET `/api/admin/dashboard/trends`

Query：`metric=users|memories|public_memories`，`days=7|30`（默认 7）。

```json
{
  "success": true,
  "data": {
    "metric": "memories",
    "days": 7,
    "points": [
      { "date": "2026-05-19", "count": 0 },
      { "date": "2026-05-20", "count": 3 }
    ]
  }
}
```

**SQLite 示例**（`createdAt` 按日）：

```sql
SELECT strftime('%Y-%m-%d', createdAt) AS date, COUNT(*) AS count
FROM Memory
WHERE deletedAt IS NULL
GROUP BY date
```

返回前补齐连续日期，无数据日 `count: 0`。

`public_memories`：`isPublic = 1` 且 `deletedAt IS NULL`。

## 4. 管理端页面 `/dashboard`

- Row1：4～6 个指标卡（`t-card` + 数字/statistic 文案）
- Row2：折线图（`echarts` + `vue-echarts`，与 C 端技术栈兼容；MVP 也可用 `t-table` 按日展示）
- Row3（可选）：最近 10 条印记 `t-table`，列：标题、用户、时间 → 点击进 M03

登录成功默认 `router.push('/dashboard')`（M01 登录页跳转此处）。
