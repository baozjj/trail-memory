## 3. 数据模型

```prisma
enum AdminRole {
  SUPER_ADMIN
  OPERATOR
  VIEWER
}

enum AdminStatus {
  ACTIVE
  DISABLED
}

model AdminUser {
  id           String      @id @default(cuid())
  email        String      @unique
  passwordHash String
  displayName  String
  role         AdminRole
  status       AdminStatus @default(ACTIVE)
  lastLoginAt  DateTime?
  createdAt    DateTime    @default(now())
  updatedAt    DateTime    @updatedAt
  auditLogs    AuditLog[]  // M07 关联，可先不加 relation 待 M07 迁移
}
```

**种子**（`prisma/seed.ts`，仅 dev）：

- email: `admin@trail.local`
- password: 随机 12 位，控制台打印
- role: `SUPER_ADMIN`

## 4. 接口

### POST `/api/admin/auth/login`

Body：

```json
{ "email": "admin@trail.local", "password": "xxx" }
```

成功：

```json
{
  "success": true,
  "data": {
    "token": "<jwt>",
    "admin": {
      "id": "...",
      "email": "admin@trail.local",
      "displayName": "超级管理员",
      "role": "SUPER_ADMIN",
      "status": "ACTIVE"
    }
  }
}
```

失败：401，`message`: 「邮箱或密码错误」（邮箱不存在 / 密码错 / DISABLED 均相同）。

**限流**：该路由单独 `rateLimit` 15 分钟 20 次/IP。

### GET `/api/admin/auth/me`

需 `adminAuthMiddleware`。

### POST `/api/admin/admins`（SUPER_ADMIN）

Body：`email`、`password`（≥8）、`displayName`、`role`。

### GET `/api/admin/admins`（SUPER_ADMIN）

分页，字段：`id`、`email`、`displayName`、`role`、`status`、`lastLoginAt`、`createdAt`。

### PATCH `/api/admin/admins/:id`（SUPER_ADMIN）

Body 可选：`displayName`、`role`、`status`。禁止将**自己**设为 DISABLED。

### POST `/api/admin/admins/:id/reset-password`（SUPER_ADMIN）

Body：`password`（≥8）。写审计 `ADMIN_RESET_PASSWORD`（M07）。

## 5. 后端实现要点

### 5.1 JWT（勿复用 `lib/jwt.ts` 的 C 端载荷）

新建 `backend/src/lib/admin-jwt.ts`：

```ts
export interface AdminJwtPayload {
  adminUserId: string;
  email: string;
  role: AdminRole;
}
```

使用 `ADMIN_JWT_SECRET`、`ADMIN_JWT_EXPIRES_IN`。

### 5.2 `adminAuthMiddleware`

解析 Bearer → `verifyAdminToken` → `req.adminAuth = { adminUserId, email, role }`。

校验 DB 中 `status === ACTIVE`（可选二次校验，防禁用后 token 仍有效）。

### 5.3 `requirePermission(permission: string)`

| role        | 规则                                              |
| ----------- | ------------------------------------------------- |
| SUPER_ADMIN | 全部允许                                          |
| OPERATOR    | 拒绝 `admin.user.*`、`settings.write`             |
| VIEWER      | 仅允许 `*.read` 和 `audit.read`、`dashboard.read` |
