## 3. 数据模型

```prisma
model AuditLog {
  id          String    @id @default(cuid())
  adminUserId String
  adminUser   AdminUser @relation(fields: [adminUserId], references: [id])
  action      String
  targetType  String
  targetId    String
  payload     String    @default("{}")
  ip          String?
  createdAt   DateTime  @default(now())

  @@index([adminUserId])
  @@index([action])
  @@index([createdAt])
}
```

## 5. `writeAuditLog` 实现

```ts
// backend/src/admin/shared/audit.ts
export interface WriteAuditLogParams {
  adminUserId: string
  action: string
  targetType: string
  targetId: string
  payload?: Record<string, unknown>
  ip?: string
}

export async function writeAuditLog(params: WriteAuditLogParams): Promise<void> {
  await prisma.auditLog.create({
    data: {
      adminUserId: params.adminUserId,
      action: params.action,
      targetType: params.targetType,
      targetId: params.targetId,
      payload: JSON.stringify(params.payload ?? {}),
      ip: params.ip ?? null,
    },
  })
}
```

**调用约定**：

- 在 service **业务成功之后**调用
- `adminUserId` 来自 `req.adminAuth.adminUserId`
- `ip` 来自 `req.ip` 或 `req.headers['x-forwarded-for']` 首段

### GET `/api/admin/audit-logs`

| Query | 说明 |
|-------|------|
| `adminUserId` | 操作人 |
| `action` | 精确匹配 |
| `targetType` | 精确匹配 |
| `targetId` | 精确匹配 |
| `from` / `to` | ISO datetime |
| `page` / `pageSize` | 分页 |

Item：

```ts
{
  id: string
  createdAt: string
  adminUserId: string
  adminDisplayName: string
  adminEmail: string
  action: string
  targetType: string
  targetId: string
  ip: string | null
  payload: Record<string, unknown>
}
```

默认排序：`createdAt desc`。
